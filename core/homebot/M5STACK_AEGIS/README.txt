# AEGIS HomeBot for M5Stack CORE2
# ============================
# 
# INSTALLATION:
# 1. Open M5Burner app
# 2. Select "MicroPython" category
# 3. Download any MicroPython firmware for M5Stack CORE2
# 4. Or use the generic ESP32 MicroPython firmware below
#
# Then upload this main.py via:
# - M5Burner's "Custom" tab
# - Or Thonny IDE
# - Or ampy/rshell tools
#
# CONNECTION:
# - Serial: COM4 (115200 baud)
# - Or WiFi (if configured)
#
# COMMANDS:
# FWD  - Move forward
# REV  - Move backward
# LFT  - Move left
# RGT  - Move right
# RTL  - Rotate left
# RTR  - Rotate right
# STP  - Stop
# PING - Check connection
# STATUS - Get status
# INFO - Device info
# HELP - Show commands
